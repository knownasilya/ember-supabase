ember-supabase
==============================================================================

Ember Data and Ember Simple Auth integrations for [Supabase][https://supabase.com/].


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.16 or above
* Ember CLI v2.13 or above
* Node.js v10 or above


Installation
------------------------------------------------------------------------------

```
ember install ember-supabase
```
